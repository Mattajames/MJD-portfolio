Netbiscuits Visitor Flow Web App

Design | Development | Mobile

Visitor Flow goes beyond traditional analytics by clearly showing which content drives the highest traffic, broken down by multiscreen categories. It shows how visitors move through your website and how engagement changes across subsequent pages and content types. In a nutshell, Visitor Flow unlocks the data to identify what’s working and what isn’t and helps you quickly fix multiscreen user experience problems.

Wireframes and Research

I worked with the Product Manager and Java developers and come up with wireframes and designs. The challenge was to set out the information architecture and present the data in a logical way that was easy for the user to read. My task was to research the best Front End technologies to plot the sankey diagrams to fit the design. I then proceeded to make prototypes to present to the team in working HTML CSS & JS View wire Frames & presentation
